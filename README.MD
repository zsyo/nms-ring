# 为超级行星探针程序添加提示音

## 1.要求

- 当前版本对应 超级行星探针V3.2.6.12.2
- 当前程序可能需要满足系统版本为win10版本以上才能正常使用,更低版本系统未测试

## 2.使用方法

1. 进入解压后的目录
2. 在该目录中鼠标右键-在终端中打开
3. 在终端中是用以下命令运行
	./nms-ring.exe "超级行星探针.exe的路径(如路径中有空格,必须在完整路径外面添加英文引号,可以直接拖动超级行星探针.exe到终端中自动粘贴路径)"
	
	如:    `./nms-ring.exe "E:\Tool\Crack\无人深空\超级行星探针.exe"`
4. 当运行时发出"警告,侦测到<LEVEL>级乐土"的声音提示后,说明当前星系有最高<LEVEL>等级的乐土星

## 3.自定义提示铃声

1. 如果没有自定义铃声,程序将使用默认的提示音
2. 当前程序仅支持ogg、wav和mp3三种格式的音频文件
3. 将音频文件改名为`customRing.ogg`、`customRing.wav`或`customRing.mp3`
4. 将音频文件放置在解压后的目录中,并确保文件名和格式匹配
5. 重新运行该程序

## 4.自定义提示等级

1. 如果没有自定义提示等级,程序将默认使用S作为提示等级
2. 当前程序支持的提示等级(从低到高)为: E、D、C、B、A、S、SS、SS+、SSS, 共9种等级
3. 设置提示等级后,程序将只在到达该等级或更高等级的时候进行提示,比如设置为A级,则B级或更低等级将不会进行提示,A级或更高等级才会进行提示
4. 自定义等级方式为运行时添加参数 `-l=<LEVEL>`, 如:

  `./nms-ring.exe -l=A "E:\Tool\Crack\无人深空\超级行星探针.exe"` 或
  
  `./nms-ring.exe "E:\Tool\Crack\无人深空\超级行星探针.exe" -l=A`
